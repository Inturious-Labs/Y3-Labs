# Fighter Jet Game

A side-scrolling shooter game where you control a fighter jet and destroy enemy aircraft.

## Features

- **Player Controls**: Move your fighter jet with WASD or Arrow keys
- **Combat**: Shoot bullets with the SPACE bar
- **Enemies**: Enemy aircraft spawn from the top and move down
- **Scoring**: Earn points by destroying enemies
- **Game Over**: Avoid collisions with enemies or it's game over!

## How to Play

**Simply open `index.html` in your web browser!** No installation needed.

1. Double-click `index.html` or open it in any modern web browser (Chrome, Firefox, Safari, Edge)

2. **Controls**:
   - **WASD** or **Arrow Keys**: Move your fighter jet
   - **SPACE**: Shoot bullets
   - **R**: Restart after game over

3. **Objective**: 
   - Destroy as many enemy aircraft as possible
   - Avoid colliding with enemies
   - Try to achieve the highest score!

## Game Mechanics

- Enemy aircraft spawn randomly from the top of the screen
- Bullets travel upward and destroy enemies on contact
- Colliding with an enemy ends the game
- Score increases by 10 points for each enemy destroyed

Enjoy the game!
